# Personal Tutoring Tracking System (PTTS)

## Overview
The Personal Tutoring Tracking System (PTTS) is a lightweight Management Information System (MIS) designed to support universities in managing and monitoring personal tutoring activities. The system enables tutors to log meetings, track referrals, and monitor student engagement, while providing departmental administrators with secure oversight and reporting tools.

This project was developed as part of the CIS4509 Advanced Professional Practice module (Level 7), using a Hybrid Waterfall–Agile SDLC to balance structured documentation with iterative development.

---

## Project Objectives
- Improve the consistency and accuracy of personal tutoring records.
- Provide a centralised platform for tutors and administrators.
- Support early identification of students requiring additional support.
- Reduce fragmentation caused by spreadsheets, emails, and manual notes.
- Explore commercialisation potential for a scalable tutoring MIS.

## Technologies Planned
- **Python** – Core programming language  
- **SQLite** – Lightweight database for rapid prototyping

## System Features (Planned & Implemented)
### Tutor Features
- Login authentication  
- View assigned tutees  
- Log tutoring meetings  
- Record referrals and recommendations  

### Admin Features
- View all tutor–student interactions  
- Monitor compliance and engagement  
- Generate oversight reports  

---

## SDLC Approach
The project followed a **Hybrid Waterfall–Agile SDLC**:

### Waterfall Phases
- Requirements gathering  
- Stakeholder analysis  
- Feasibility study  
- System design (UML, ERD, wireframes)

### Agile Sprints
- Backend development  
- Frontend development  
- Integration  
- Testing and refinement  

This approach allowed structured planning while enabling iterative improvements.
